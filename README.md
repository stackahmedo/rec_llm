🎙️ RecLLM（レック・エルエルエム）

AI-Powered Desktop Transcription & Meeting Analysis Software

AI音声文字起こし・会議分析デスクトップソフトウェア

RecLLM is a professional Windows desktop application for transcription, speaker analysis, meeting summarization, and PDF report generation.

RecLLM は、音声文字起こし・話者分析・会議要約・PDFレポート生成を行う、Windows向けの業務用AIデスクトップアプリケーションです。
